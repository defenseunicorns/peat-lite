# Code of Conduct

Community members are required to abide by the [OpenSSF Code of Conduct](https://openssf.org/community/code-of-conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing [conduct@defenseunicorns.com](mailto:conduct@defenseunicorns.com).

All reports will be reviewed and investigated promptly and fairly. The project team is obligated to maintain confidentiality with regard to the reporter.
